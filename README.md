# Gist Download Extension

Chrome extension with ReactJS to download Github Gist.

![Capture ext](./.readme-static/capture.png)

## Build Setup

``` bash
# install dependencies
npm install

# watching code
npm run watch

# build for production with minification
npm run build
```
